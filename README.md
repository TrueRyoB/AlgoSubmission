### レポジトリ概要
- todo.txt: 精進の予定などを記しています
- lib/test.cpp: テストデータの生成に役立ちます
- lib/whine.cpp: 解説ACでWAを取るためのメソッドです
- lib/*.cpp: その他のライブラリです

### 訂正
- submissionという名目とは裏腹に、提出ファイルを保管はしていません。
- [AtCoder Problems](https://kenkoooo.com/atcoder/#/table/)様を通じて、適切に提出履歴の管理ができるためです。
- 都度の新規ファイル生成には[nomouse-cli](https://github.com/TrueRyoB/nomouse-cli)を活用しています。
